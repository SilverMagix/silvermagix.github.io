Works

Goblin Dodger -2018-
A 2D solo freshman year game created with Monogame in C# completed in a month. It consists on dodging goblins coming from different sides while jumping on platforms. This was my first project made during my program.  I implemented its physics, character movement, background layers, enemies’ behavior, level layout and UI.

Element Hunter  -2019-
  A 3D action game where your goal is to kill the dragon. It was a 2 man effort made during a four month period using UE4 as an introduction to the engine. I took responsibility on implementing the dragon. I designed its behavior, its attacks and movement. I also helped tweak the player character’s ranged attack. I also set-up a Perforce server based on CentOS 7 to help with source control.

Mecanica –2020-
A 2.5D parkour action game made in UE4. This is a solo-project made in my last year of CEGEP. I did some 3D modeling for my character using Blender and some assets for level designing. I used Perforce as a source control option until I switched to GitHub for demonstration purposes. I programmed every gameplay feature and designed the levels.
